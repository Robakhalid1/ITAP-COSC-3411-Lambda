# ITAP-COSC-3411-Lambda

# Mini Shell in Python - Lambda

A simple command-line shell implemented in Python that executes basic system commands and provides built-in utilities.

# Prerequisites
- Python 3.6 or higher
- No additional packages required (uses only Python standard library)


# Installation
1. **Clone the repository**:
   bash
   git clone: https://github.com/Robakhalid1/ITAP-COSC-3411-Lambda
   cd Alpha

 # Configuration
No additional configuration required. The shell works out-of-the-box using only Python standard library.

# Running the Tool
Execute the shell with: python3 mini_shell.py
