# Mini Shell in Python - Lambda

A simple command-line shell implemented in Python that executes basic system commands and provides built-in utilities.

# Features

- Built-in Commands: `cd`, `pwd`, `echo`, `help`, `clear`, `exit`, `quit`
- External Command Support: Execute any system command like `ls`, `cat`, `mkdir`, etc.
- Error Handling: Proper error messages for common issues
- Cross-platform: Works on Windows, macOS, and Linux

# Installation

# Prerequisites
- Python 3.6 or higher
- No additional packages required (uses only Python standard library)
